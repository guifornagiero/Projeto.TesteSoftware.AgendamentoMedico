Rodar testes unitários:
`dotnet test .\Unitarios\Unitarios.csproj`

Rodar testes de exception:
`dotnet test .\Exceptions\Exceptions.csproj`

Rodar testes de integração
`dotnet test .\Integration\Integration.csproj`
